# vscode fecs插件
fecs extension for vscode
# install
## install fecs
```
sudo npm install fecs -g
```
## install fecs extension for vscode
in vscode,press ```cmd + shift + p```
```
ext install fecs
```

# use 
open a js、HTML or css file


press ```cmd + shift + p```

type
```
fecs
```
then you can see the fecs output
